 A Python-based movie rental system utilizing OOP principles to manage inventory, customer accounts, and rental transactions.

 # Movie Rental Application

A lightweight, terminal-based Python application designed to manage movie 
inventories, track customer rentals, and calculate late fees dynamically.

## Features
* **User Authentication:** Secure login for administrators and customers.
* **Inventory Management:** Real-time tracking of available DVD and digital titles.
* **Rental System:** Automated checkout, return processing, and receipt generation.
* **Pricing Engine:** Dynamic fee calculation based on rental duration and movie type.
